XPath3.1とXSLT3.0時代が来る！

JSONからある特定のデータを抽出したい場合、どうしていますか。
JSONは、Webアプリケーションでよく使用されるデータ形式ですが、そのデータから必要な情報を取り出すことは容易ではありません。
しかし、実は最新のXPath3.1ならJSONから高度な抽出を行うことができます。

{
    "test":"val",
    "arr":[
        {
            "n":"v1",
            "n2":1
        },
        {
            "n":"v2",
            "n2":2
        },
        {
            "n":"v3",
            "n2":3
        }
    ],
    "map":{
        "test2":"test"
    }
}

このJSONをjsonという変数にバインドして、arrの中からn2の値が2以上のマップからnの値を抜き出す場合、次のようなコードになります。

let $json := parse-json($defs?json)
$json?arr?*[?n2 >= 2]?n
